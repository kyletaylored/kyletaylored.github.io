Sharp-cornered, uppercase CTA button. Use for all primary and secondary actions.

```jsx
<Button label="Browse Experiments" variant="primary" icon="→" />
<Button label="View All Tools" variant="secondary" />
<Button label="View on GitHub" variant="dark" />
```

Variants: `primary` (teal fill), `secondary` (black outline), `ghost` (no border), `dark` (white outline on dark bg), `dark-solid` (teal on dark bg).
Sizes: `sm`, `md` (default), `lg`.
